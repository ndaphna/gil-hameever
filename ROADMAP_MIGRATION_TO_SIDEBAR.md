# 🗺️ העברת תפריט "מפת דרכים" לסיידבר

## 🎯 מטרה

העברת כל תוכני "מפת דרכים" לסיידבר הפנימי, כך שיהיו נגישים **רק למשתמשות רשומות** (בעלות מנוי או בתקופת ניסיון).

---

## ✅ מה בוצע?

### 1. **הוספת תפריט מפת דרכים לסיידבר** ✨
📁 `src/app/components/Sidebar.tsx`

**נוסף:**
- State חדש: `roadmapOpen` לניהול פתיחת/סגירת התפריט
- רשימה חדשה: `roadmapItems` עם כל 6 השלבים של מפת הדרכים
- תפריט נפתח חדש בתוך `.sidebar-nav`

**כולל:**
```javascript
const roadmapItems = [
  { href: '/menopause-roadmap', icon: '🗺️', label: 'מפת הדרכים המלאה' },
  { href: '/the-body-whispers', icon: '🧏🏻‍♀️', label: 'שלב 1: הגוף לוחש' },
  { href: '/certainty-peace-security', icon: '🌳', label: 'שלב 2: וודאות, שקט, ביטחון' },
  { href: '/belonging-sisterhood-emotional-connection', icon: '🤝', label: 'שלב 3: שייכות ואחוות נשים' },
  { href: '/self-worth', icon: '🌟', label: 'שלב 4: ערך עצמי, משמעות' },
  { href: '/wisdom-giving', icon: '✨', label: 'שלב 5: תבונה ונתינה' }
];
```

---

### 2. **עיצוב התפריט הנפתח בסיידבר** 🎨
📁 `src/app/components/Sidebar.css`

**נוסף:**
- `.sidebar-dropdown` - מכיל תפריט נפתח
- `.sidebar-dropdown-toggle` - כפתור פתיחה עם חץ מסתובב
- `.sidebar-dropdown-menu` - התפריט הנפתח עצמו עם אנימציה
- `.sidebar-dropdown-item` - פריט בודד בתפריט
- אנימציית `slideDown` חלקה

**תכונות:**
- חץ שמסתובב 180° בלחיצה
- רקע עדין ורוד (`rgba(255, 0, 128, 0.03)`)
- Hover effects עם הדגשה ורודה
- Active state לדף הנוכחי

---

### 3. **הסרה מה-Navigation הראשי** 🚫
📁 `src/app/components/Navigation.tsx`

**הוסר:**
- State: `isRoadmapOpen`
- Function: `toggleRoadmap()`
- כל הבלוק של:
  ```jsx
  <div className={`nav-dropdown ${isRoadmapOpen ? 'active' : ''}`}>
    {/* כל התפריט הנפתח */}
  </div>
  ```

**תוצאה:**
ה-Navigation עכשיו מכיל רק:
- דף הבית
- אודות
- מחירים
- אזור אישי (למשתמשות מחוברות)

---

### 4. **עדכון דף הבית** 🏠
📁 `src/app/page.tsx`

**שונה ב-Footer:**
```jsx
// לפני:
```

## ✨ תכונות נוספות

### אנימציות:
1. **slideDown** - התפריט הנפתח נפתח בצורה חלקה

- ניווט מקלדת מלא

- Semantic HTML

### Responsive:
- Desktop: הסיידבר תמיד פתוח בצד
