## 🎨 מאפיינים עיקריים

### 1. **רקע גרדיאנט דינמי**
```css
background: linear-gradient(135deg, 
  #E91E63 0%,    /* ורוד פוקסיה */
  #9C27B0 50%,   /* סגול */
  #673AB7 100%   /* סגול כהה */
);
```

**אנימציה:**
- הגרדיאנט זז באיטיות (15 שניות)
- יוצר תחושה חיה ודינמית
- לא מציף, מאוד עדין

### 2. **טיפוגרפיה חזקה**
```
גודל: 2.8rem - 6.5rem (responsive)
משקל: 900 (Ultra Bold)
צבע: לבן
letter-spacing: -0.02em (Tight)
```

**שתי שורות:**
1. "גיל המעבר," - משקל 400 (Light)
2. "בצורה מעצימה" - משקל 900 (Bold)

### 3. **תת-כותרת רגועה**
```
גודל: 1.1rem - 1.5rem
צבע: לבן עם opacity 0.95
קריא ומזמין
```

### 4. **חץ גלילה אנימטיבי**
- קופץ בעדינות (bounce animation)
- לחיצה = גלילה חלקה למטה
- Hover effect: scale(1.1)
- נגיש מלא (aria-label + focus state)

## 📱 Responsive Design

### 🖥️ Desktop (1024px+)
- כותרת: עד 6.5rem
- גובה מלא: 100vh
- חץ גדול: 40x40px

### 📱 Tablet (768px-1024px)
- כותרת: 4-5rem
- מרווחים מותאמים
- חץ: 40x40px

### 📱 Mobile (<640px)
- כותרת: 2.2-3.5rem
- חץ קטן יותר: 32x32px
- טקסט מותאם לרוחב קטן

## ⚡ אנימציות

### 1. **fadeInUp** (כניסת תוכן)
```css
from: opacity 0, translateY(30px)
to: opacity 1, translateY(0)
duration: 1s
```

### 2. **fadeIn** (כותרת ותת-כותרת)
```css
duration: 0.8s
staggered: 0.2s, 0.4s, 0.6s
```

### 3. **gradientFlow** (רקע)
```css
background-position: 0% → 100% → 0%
duration: 15s
easing: ease
infinite loop
```

### 4. **bounce** (חץ גלילה)
```css
translateY: 0 → -10px → 0
duration: 2s
infinite loop
```
