## 📱 בדיקת Responsive

### Desktop (רוחב מעל 1024px)
- פתח את DevTools (F12)
- בחר Responsive Mode
- הגדר רוחב: 1920px
- וודא שהכל נראה מעולה

### Tablet (768px - 1024px)
- הגדר רוחב: 768px
- וודא grid של 2 עמודות

### Mobile (מתחת 768px)
- הגדר רוחב: 375px
- וודא grid של עמודה אחת
- בדוק תפריט המבורגר

## 🐛 בעיות נפוצות ופתרונות

### האתר לא נטען?
```bash
# נסה לעצור ולהפעיל מחדש
npm run dev
```

### העיצוב לא נראה טוב?
- נקה cache של הדפדפן (Ctrl + Shift + R)
- וודא שקובץ `home.css` נטען

### האנימציות לא עובדות?
- בדוק אם יש שגיאות בקונסול (F12)
- וודא שהדפדפן תומך ב-CSS animations

## 🎨 התאמות אפשריות

רוצה לשנות משהו? הקבצים לעריכה:

```
src/app/page.tsx        → תוכן HTML של הדף
src/app/home.css        → כל העיצוב
```

### דוגמאות לשינויים קלים:

**שנה צבעים:**
```css
/* בקובץ home.css, שנה את המשתנים */
--magenta: #FF0080;
--purple: #9D4EDD;
```

**שנה גודל טקסט:**
```css
.hero-title {
  font-size: clamp(3rem, 8vw, 6rem); /* הגדל/הקטן */
}
```

**הוסף תמונת רקע:**
```css
.home-hero {
  background-image: url('/path/to/image.jpg');
  background-size: cover;
}
```
