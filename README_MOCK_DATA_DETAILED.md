## 🎨 **דוגמאות לשימוש**:

### **בדיקת דפוסי שינה**
```sql
SELECT date, sleep_quality, energy_level 
FROM daily_entries 
WHERE user_id = 'user-id' 
ORDER BY date;
```

### **בדיקת תסמינים**
```sql
SELECT date, hot_flashes, night_sweats, mood 
FROM daily_entries 
WHERE user_id = 'user-id' 
AND hot_flashes = true;
```

### **בדיקת קשרים**
```sql
SELECT sleep_quality, COUNT(*) 
FROM daily_entries 
WHERE user_id = 'user-id' 
GROUP BY sleep_quality;
```
