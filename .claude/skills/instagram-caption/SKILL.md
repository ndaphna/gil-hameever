# סקיל: קפשן אינסטגרם — גיל המעבר

## לפני הכל

קרא את `<PROJECT_ROOT>\CLAUDE.md` — שם כללי המותג, שפה, אווטאר "מיכל".

## מה הופך קפשן לטוב?

באינסטגרם רואים רק **שורה אחת** לפני כפתור "קרא עוד".
זו שורת ה-Hook — ההכל.

קפשן טוב עושה שני דברים:
1. עוצר את הגלילה בשורה הראשונה
2. עומד לבד גם בלי הקרוסלה

## קלט נדרש

1. **נושא הקרוסלה** — מה הנושא? (חובה)
2. **script.md** — קראי אותו לפני הכתיבה (לשמור על עקביות)
3. **קול ראשי** — ענבל / עליזה / שניהם (ברירת מחדל: ענבל, עם קריצה של עליזה)

---

## מבנה הקפשן — 6 שכבות

### שכבה 1: שורת Hook (חובה — קריטי)

שורה אחת בלבד. ייעוד: לעצור בגלילה.

**עקרון:** היא רואה רק את השורה הזו. אם היא לא תלחץ "קרא עוד" — הכל לשווא.

**שיטות לשורת hook:**

| שיטה | דוגמה |
|------|-------|
| שאלה שהיא שואלת את עצמה | "רק אני מרגישה שהגוף שלי החליף חוקים בלי להודיע לי?" |
| הצהרה שמרגישה "זה אני" | "יש ימים שהמוח פשוט לא שם. ולא מסבירים לך למה." |
| קונטרסט מפתיע | "ישנתי 8 שעות ועדיין עייפה. הגיע הזמן להבין למה." |
| וידוי קטן | "לפני שבוע שכחתי מילה בפגישה. עמדתי שם ולא מצאתי אותה." |
| עובדה שמפילה אסימון | "60% מהנשים בגיל המעבר חוות ערפל מוחי. כולן חשבו שזה רק עייפות." |

**כללים לשורת hook:**
- לא להתחיל ב"אני"
- לא שאלה רטורית "יפה" — שאלה שכואבת
- לא קלינית ("תסמינים", "הורמוני")
- לא ארוכה — משפט אחד
- אסור מקף ארוך (—)

---

### שכבה 2: ריווח (שורה ריקה)

תמיד שורה ריקה אחרי ה-hook. זה "אוויר" שמזמין להמשיך.

---

### שכבה 3: גוף — הסיפור (3–6 שורות)

קצר. נושם. כל משפט חשוב — שורה משלו.

**מבנה מומלץ:**
```
[סיטואציה יומיומית — גובה עיניים]

[רגע אישי — הקורא מרגיש "היא מדברת עליי"]

[נפילת אסימון — תובנה אחת, פשוטה]
```

**עקרונות שפה (מהמדריך):**
- גוף ראשון + פנייה ישירה ("את")
- ביטויים נכונים: "מכירה את הימים...", "פתאום...", "ואז נפל לי האסימון", "זה לא את, זה..."
- ביטויים אסורים: "גיל הבלות", שפה רפואית, "סליחה שאני ככה", "אין מה לעשות"
- אמוג'ים: עד 2-3 — מהרשימה: 🌿 ❤️ 👑 💙 ✨
- שלוש נקודות... ליצירת השהייה רגשית
- Bold על משפטי מחץ (1-2 בקפשן)

**טרנספורמציה:**
הגוף חייב לנוע:
`קושי / ענן` → `בלבול` → `נפילת אסימון` → `הכוח`

**קריצת עליזה (אופציונלי):**
אם הנושא כבד מדי, אפשר להוסיף משפט קצר בטון עליזה:
"(עליזה הייתה אומרת: 'מותק, זה לא אלצהיימר. זה גיל המעבר. ברוכה הבאה.')"

---

### שכבה 4: CTA — הנחייה לפעולה

**עיקרון:** הנחייה אחת בלבד. רכה אבל ברורה.

**סוגים:**
| סוג | דוגמה |
|-----|-------|
| שיתוף | "שתפי עם חברה שחשבה שהיא לבד בזה." |
| שמירה | "שמרי את הפוסט הזה — ותחזרי אליו ביום קשה." |
| תגובה | "כתבי לי בתגובות: ✋ אם גם את מכירה את הרגע הזה." |
| שאלה מזמינה | "ספרי לי — מה עזר לך להתמודד עם ערפל מוחי?" |

**אסור ב-CTA:**
- "לחצי כאן"
- "לרכישה"
- "לינק בביו" (אלא אם מבקשים במפורש)
- יותר מהנחייה אחת

### שכבה 5: שורה ריקה + סגירה

לפני ההאשטאגים, אפשר להוסיף משפט DNA אחד:

```
לא "גברת". גיבורה. 👑
```

או:
```
את לא לבד בזה. 🌿
```

### שכבה 6: האשטאגים

**קבועים (לכל קפשן):**
```
#גילהמעבר #גיל_המעבר #לאגברתגיבורה #ענבלדפנה #גיבורה
```

**נושאיים (לפי תוכן) — בחרי 5-8 מתוך:**
```
#ערפלמוחי #זיכרון #ריכוז          ← ערפל מוחי
#שינה #שינהטובה                   ← שינה
#אנרגיה #עייפות                   ← אנרגיה
#גלוחום #הורמונים                 ← גלי חום
#ביטחון #זהות #גיבורה             ← העצמה
#אישה50 #אישה45 #גיל50           ← גיל
#גופשלי #בריאות                   ← גוף
```

**כמות כוללת:** 10-15 האשטאגים. לא יותר.

## מתודולוגיית עריכה (מהמדריך)

1. **פתחי בהזדהות** — משפט שמרגיש כמו שיחה (שאלה / סיטואציה מוכרת)
2. **פנייה ותהודה** — "את", שאלות שמעוררות הנהון
3. **קיטוע ואוויר** — משפטים קצרים, כל אחד בשורה משלו
4. **טרנספורמציה** — מקושי לכוח, "נפילת אסימון"
5. **הומור/ציניות** — אם כבד מדי, קריצת עליזה
6. **הדגשות** — Bold על 1-2 משפטי מפתח

## כללי ברזל

1. **שורה ראשונה = הכל.** בדקי שהיא עובדת לבד.
2. **אל תשתמשי במקף ארוך (—)** — אסור לחלוטין
3. **לא "גיל הבלות"** — לעולם לא
4. **לא שפה קלינית** — "תסמינים", "הורמונלי"
5. **לא AI** — אם זה נשמע כמו ChatGPT, כתבי מחדש
6. **Bold רק על 1-2 משפטים** — לא יותר

## תזרים עבודה

1. קראי `script.md` של הקרוסלה
2. זהי את הרגע הרגשי המרכזי (לא את הטיפ — את הרגש)
3. כתבי 2-3 גרסאות לשורת ה-Hook — בחרי את החדה ביותר
4. כתבי את הגוף — קצר, נושם, עם נפילת אסימון
5. סיימי ב-CTA אחת + האשטאגים
6. שמרי ל-`caption.md`
7. דווחי: "קפשן מוכן — [נושא], Hook: [השורה]"
